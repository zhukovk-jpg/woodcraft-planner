Задачи КМ

📋 Приёмка образцов

Образцы принимаются только на датах приёмки. Изделия попадают в список автоматически по дате завершения этапа образцов. Руководитель отмечает результат — он влияет на дальнейший план.

🔨 Участок сборки КМ

На сборку попадают изделия в последний день изготовления образца и в последний день нулевой партии. Конструктор или руководитель отмечают результат.

Дополнительные задачи

Задачи выдаются конструкторам как доп. работа. Оплата — после полного выполнения всех этапов.

☁️ Онлайн-синхронизация

Введите URL вашей Firebase Realtime Database для синхронизации данных в реальном времени между всеми пользователями.

Пример: https://np-planner-default-rtdb.firebaseio.com
Как настроить (бесплатно):
1. Зайдите на console.firebase.google.com
2. Создайте проект → Realtime Database → «Начать в тестовом режиме»
3. Скопируйте URL базы (вида https://...firebaseio.com) и вставьте выше
